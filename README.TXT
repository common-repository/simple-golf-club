A simple interface to manage a small golf club.

== Description ==
Simple Golf Club provides an easy way you to coordinate golf outings for your friends or for large groups of people.

Create teams, add players, schedule events, and track player scores using a simple interface.

For more information, check out the [SGC Website](https://simplegolfclub.com/)

For more information and to contact the developers, checkout [GitLab](]https://gitlab.com/mlinton/simplegolfclub/).

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/simplegolfclub` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Simple Golf Club screen to configure the plugin

== Screenshots ==
1. Events: Create events for a team at a specific Location. Divide teams into smaller groups for easier round management.

2. Locations: Keep track of location information including tee statistics for each course.

3. Teams: Create teams, and keep track of the players on that team.

4. Players: Add players and assign them to one or more teams.

5. Score Cards: For each event keep track of player stats by filling in score cards.

6. Settings: Set defaults and configure the behavior of Simple Golf Club.
